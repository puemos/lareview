01

AI-Powered Planning

Input a PR reference or paste a raw diff. LaReview uses your coding agent to analyze intent and generate a structured review plan.

02

Task-Focused Diffs

Stop context switching. Select a task, and the viewer isolates only the relevant hunks of code. Review step-by-step.

03

Agent Feedback

The AI proactively identifies bugs and style issues, creating feedback threads anchored to specific lines of code automatically.

04

Local Context

Link local Git repos to give the agent full access to search your codebase, providing deeper insights than a simple diff review.

05

GitHub Sync

Submit your review feedback directly to GitHub PRs with automatic summary generation.

06

CLI Support

Launch reviews directly from your terminal. Load PRs or pipe a git diff

$ lareview
$ lareview pr owner/repo#123
$ git diff | lareview
$ lareview --agent claude
07

Export & Share

Export your review summary to Markdown with diagrams and code insights. Copy to clipboard or save to file.

Workflow

INPUT

Input

Paste a unified diff or a GitHub PR URL (e.g., owner/repo#123).

FETCH

Fetch

LaReview fetches data locally via the GitHub CLI (`gh`). No intermediate servers.

GENERATE

Generate

Your chosen coding agent (opencode, claude code, codex, etc.) analyzes intent and builds a task tree.

REVIEW

Review

Execute the plan. Add notes, track status, and export to Markdown.
